# Notion
###### v 0.0.2
## About
Building a Notes taking service prototype. Inspired by Notion app. The project saves data using localstorage techology.
Functionality:
* Edit main title
* Edit subtitles
* Edit text blocks
* Add and delete new subtitles and text blocks

## Technologies
* HTML
* CSS
* JS
